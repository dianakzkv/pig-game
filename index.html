About the game

❓Dice roll button change the number of points on the die. Hold button add your current score to total scores. New game button resets the game.
❗If a player gets one point on the die, the current score is reset to 0 and the turn passes to the next player.
🎉The winner of this game must score 💯 points or more!

Enjoy the game and have fun! Good luck 😉
